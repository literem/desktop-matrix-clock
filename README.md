# 16x40像素点的WiFi网络点阵时钟

这是一个用ESP8266制作的WiFi网络时钟，连接WiFi后自动获取网络时间并显示。此外，通过Qt编写的上位机软件，可以设置计时、倒计时的功能，还能将文字和图片转成16x16像素大小显示出来，可以设置汉字滚动，类似于广告牌的效果。成本低廉，制作简单，已将所需的元件的购买连接整理成BOM表。
